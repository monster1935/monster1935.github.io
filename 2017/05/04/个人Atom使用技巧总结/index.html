常用快捷键

ctrl+shift+L 设置文件类型

ctrl+shift+k 删除某行

ctrl+P, ctrl+T 工程内查找文件

ctrl+shift+P 全局搜索

ctrl+, 打开设置面板

ctrl+G 打开行号跳转面板

ctrl+\ 切换左侧菜单栏面板显示隐藏

常用插件

block comment 添加块状注释 ctrl+shift+/
atom-html-preview 实时预览html文件，避免打开浏览器进行查看html文件
