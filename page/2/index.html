关于Cookie的总结

2017-10-25

HTTP cookie(web cookie, browser cookie)是服务器发送给客户端浏览器的数据片段。客户端浏览器将保存这段数据并在下次访问同一个服务器地址时带上这段数据。通常，它被用来识别两次请求是否来自同一个浏览器，保持用户的登录状态。比如，它为无状态的HTTP协议记录了状态信息。

浅谈webpack的打包机制
